## Why Markdown

Markdown keeps portfolio updates simple. You can write a post or project note in plain text, commit it, and let the site render it automatically.

## Content Folders

This site keeps content in three folders:

- `content/projects`
- `content/blog`
- `content/achievements`

The ordering comes from `content/index.json`, so the site does not need server-side folder listing.
